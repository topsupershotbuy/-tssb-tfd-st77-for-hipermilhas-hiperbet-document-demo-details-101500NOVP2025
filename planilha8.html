Planilha 8: Requisitos Técnicos (Back-End)

O Back-End é o Motor de Milhas, responsável por toda a lógica de acumulação e gestão de dados do programa HiperMilhas.

Integrações e Lógica Principal

  • Integração Core: API de escuta (webhook/endpoint) para receber Depósitos e Apostas em tempo real.
  • Lógica de Acúmulo: Cálculo de 1 Milha / R$1,00 depositado, incluindo regras para Milhas Extras.
  • Gestão de Níveis: Lógica para armazenar, atualizar e aplicar bônus percentuais (Prata, Ouro, Diamante).

APIs e Segurança

  • API de Consulta: APIs de leitura otimizadas para o Front-End consultar Saldo, Nível, Progresso e Histórico em tempo real.
  • Módulo de Resgate Digital: API para processar o resgate e integrar com a plataforma Hiperbet (liberação instantânea de bônus).
  • Segurança: Implementação de logs detalhados para todas as transações, garantindo rastreabilidade e integridade.
