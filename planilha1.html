Planilha 1: Objetivo do Projeto

HiperMilhas - Programa de Fidelidade

O objetivo do projeto é criar um programa de fidelidade inovador para a Hiperbet, onde cada depósito e aposta gera milhas acumuláveis.

Os usuários podem resgatar suas milhas por prêmios exclusivos, bônus especiais e produtos na Loja de Milhas Hiperbet.
